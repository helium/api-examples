Script ina219.main

Example main loop for the Helium Analog Extension Board

Requires helium-script 2.x and the ina219.lua library.

This main loop takes a voltage and current reading every SAMPLE_INTERVAL, and sends voltage on port v, current on port c to Helium. In semi-hosted mode the script will also print out the readings on the console.

Usage:

    # In semi-hosted mode over USB
$ helium-script -m main.lua ina219.lua

# Upload to device over USB
$ helium-script -m main.lua ina219.lua
