Module ina219.config

The configuration abstraction layer.

Helium will soon expose a configuration API that enables you to send configuration values over the air from the API to the Atom Board.

get (key[, default=nil])
Get a configuration value

Parameters:

  • key string the key to get the config value for
  • default number or boolean the default value to return (default nil)

Returns:

    number or boolean the value stored in the configuration store. If the he.config module is not present on the device the default value is returned.
