Module ina219

INA219 Helium Analog Board This is the library class for configuration and using the INA219 part used on the Helium Analog Extension Board

Usage:

    ina219 = require("ina219")
sensor = ina219:new()
current = sensor:get_current()
voltage = sensor:get_voltage()

set_calibration_32v_2a ()
Sets calibration of the analog board.

This sets the calibration values of the analog board to support measuring voltage up to 32V and current up to 2A.

This is the default and recommended configuration.

set_calibration_32v_1a ()
Sets calibration of the analog board.

This sets the calibration values of the analog board to support measuring voltage up to 32V and current up to 1A.

set_calibration_16v_400ma ()
Sets calibration of the analog board.

This sets the calibration values of the analog board to support measuring voltage up to 16V and current up to 40mA.
